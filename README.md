# Proyecto FitnessLion

Aplicación web desarrollada en Java con Spring Boot para la gestión de usuarios, clases y reservas.

## Tecnologías utilizadas
- Java
- Spring Boot
- Thymeleaf
- MySQL
- Maven

## Instrucciones de instalación

1. Clonar o descargar el repositorio
2. Importar el proyecto en Eclipse como proyecto Maven
3. Configurar la base de datos MySQL (fitnesslion)
4. Ejecutar la aplicación (FitnesslionApplication.java)
5. Acceder desde el navegador a:
   http://localhost:8080

## Funcionalidades
- Registro de usuarios
- Listado de clases
- Reservas de clases
- Visualización de reservas
